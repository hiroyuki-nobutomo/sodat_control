3. SD カードに貼り付ける

  
    
    
  1. microSD を PC に挿し、bootfs パーティションを開く。
    2. 
    
  2. firstrun.sh をテキストエディタで開く (Win: メモ帳、Mac: テキストエディット の標準テキストモード)。
    3. 
    
  3. ファイル末尾近くにある rm -f /boot/firstrun.sh (もしくは exit 0) の直前に、上の snippet を貼り付ける (# >>> BEGIN ... >>> から # <<< END <<< までの本体行)。
    4. 
    
  4. 保存。改行コードは LF のまま (CRLF に変えない)。
    5. 
    
  5. SD カードを安全に取り外し、Raspberry Pi に挿入 → 電源 ON。
    6. 
  


  

    うまく貼れるか不安な人向け: ダウンロードしたファイルでまるごと置き換える方法
    
    
      
  1. 右の .sh としてダウンロード でファイルを保存。
    2. 
      
  2. ダウンロードした firstrun.sh をテキストエディタで開く。
    3. 
      
  3. Pi Imager が bootfs に生成した firstrun.sh を同じエディタで開き、Wi-Fi 設定や userconf 行などの上の方の中身を全部コピーして、ダウンロードしたファイルの先頭の指定箇所に貼り付ける。
    4. 
      
  4. 保存し、bootfs 上の firstrun.sh をこのファイルで上書きする。
    5. 
    

    
慣れない場合は手順 3 のシンプルな貼り付け方を推奨します。

  


  

    生成された snippet には service_account.json が base64 で含まれます。チャットやメール本文に貼らないでください。SD への書き込みが終わったらブラウザのこのタブも閉じることを推奨。

3. Place on the SD card

  
    
    
  1. Insert the microSD into your PC, open the bootfs partition.
    2. 
    
  2. Open firstrun.sh in a text editor (Notepad on Windows; TextEdit in plain-text mode on macOS).
    3. 
    
  3. Near the end you'll find rm -f /boot/firstrun.sh (or just exit 0). Immediately above that, paste the snippet body (everything between the # >>> BEGIN ... and # <<< END markers).
    4. 
    
  4. Save the file. Keep line endings as LF (don't let your editor switch to CRLF).
    5. 
    
  5. Safely eject the SD card, insert it into the Pi, power on.
    6. 
  


  

    The generated snippet embeds your service_account.json as base64. Don't paste it into chat or email. Once it's on the SD card, close this browser tab.
